# Airline-Customer-Satisfaction-Prediction
Machine Learning project to predict airline customer satisfaction using Logistic Regression, KNN, Decision Tree, Random Forest, SVM, and Naive Bayes with comprehensive model evaluation and business insights.

OVERVIEW

This project focuses on predicting airline customer satisfaction using supervised machine learning algorithms. The objective is to analyze passenger experience data and identify the factors that influence customer satisfaction.

PROBLEM STATEMENT

Airlines need to understand customer preferences and service quality factors to improve passenger satisfaction. This project uses machine learning models to predict whether a customer is satisfied or dissatisfied based on various flight and service attributes.
